Ui/UX Trend 2022-2021

Neumorphism : Trend 2019/2020.

Neumorphism (or neumorphism) is a user interface design technique of skeuomorphism (technique to imitate the appearance of
a real object when designing a virtual object). Neumorphism aims to create a combination of flat design elements coupled with skeuomorphism techniques,
making user interface elements softer and more readable. The purpose of this use is to attract more user attention. The central point of Neumorphism is
the effect of shadows applied to a form, giving this feeling of an illuminated surface.

Glassmorphism : Trend 2020/2021.

Glassmorphism is a new style of UI Design that is gaining more and more notoriety. This new trend differs from Neumorphism in several aspects:
Transparency (glass effect with a blur in the background), Using Multiple Colors with Floating Objects, Fine and sharp outline, Bright colors.
The major particularity of this trend is therefore transparency. The object of the game here is to create a hierarchy of information using
the depth of the interface. The user manages to sort and select the elements with respect to their transparency. This use of colors and transparency gives
a modern and fresh look to the interface. That's why some brands do not hesitate to use this trend, including Apple.

Claymorphism : the new trend of 2022.

Like Neumorphism, Claymorphism is based on relief effects by playing on shadows and lights, they are differentiated in the desired rendering.
While on the Neumorphime we will seek a refined and classy aspect, the Claymorphism will rather give a nice aspect, which does not take the lead,
almost a little childish. This style is used mainly by start-ups in the technology sector, and also in CRM or dashboard type interface design.
